# Instruction

#### Prepare python venv:
```
    make install_required_packages
``` 

#### Run unit tests localy:
```
    make run_tests
``` 

#### Run unit tests in docker:
```
   make build_docker
``` 
#### Test script in limited environment 
```
    make test_code_in_docker
```
#### Install scripts in local env
```
    make install_app
```
